## Why

日本のニュースを英語で聞きながら、日本語訳も同時に確認できる、完全無料で利用可能な英語リスニング学習ツールを提供します。
既存の英語ニュースアプリは有料であるか、日本の身近なニュースを扱っていないことが多いため、日本のニュースを題材にすることで背景知識を活かした効率的なリスニング学習を実現します。また、個人開発者が低コスト（無料）で持続可能な形でWeb上に公開・運用できる仕組みを構築します。

## What Changes

- **GitHub Pagesでのホスティング**: 静的サイトとして完全に無料でホスティング可能なWebアプリケーションを構築します。
- **自動ニュース配信 (Curation & Translation)**: GitHub Actionsを利用して定期的に日本のニュース（RSS）を取得し、Gemini API（無料枠）を用いて「英語のサマリー」「学習用の英文」「対訳の日本語文」を生成し、静的JSONとしてアプリに配信します。
- **インタラクティブ読み上げプレイヤー**: Web Speech API（ブラウザ標準の音声合成機能）を使用し、英語テキストの読み上げを行います。読み上げ中の英文と、対応する和文をリアルタイムに同期ハイライト表示するプレイヤーUIを提供します。
- **リスニング調整機能**: 再生速度（0.5倍〜2.0倍）の調整、音声話者（アメリカ英語、イギリス英語など）の選択機能を搭載します。

## Capabilities

### New Capabilities
- `news-curation`: GitHub ActionsおよびGemini APIを使用し、日本のニュースRSSから英語要約・対訳データを定期的に自動生成・配信するバックエンド・ビルド機能。
- `player-interface`: Web Speech APIを用いた音声合成（TTS）再生機能、および音声に同期して英語・日本語の対訳をハイライト表示するインタラクティブなプレイヤーUI。

## Impact

- **フロントエンド技術**: HTML5, Vanilla CSS, Vanilla JavaScriptのみで構築され、外部の有料バックエンドサーバーを必要としません。
- **API依存**: Google Gemini API（無料枠）およびブラウザの Web Speech API に依存します。
- **CI/CD**: GitHub Actionsでの定期スクリプト実行およびGitHub Pagesへの自動デプロイワークフローが必要となります。
