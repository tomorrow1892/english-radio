- [x] 2.2 定期的にニューススクリプトを実行し、`data/news.json` を生成して自動的にGitHub PagesへデプロイするGitHub Actionsワークフロー（`newspaper3k` 等のPython依存パッケージインストールステップを含む）を作成する。
- [x] 2.3 アプリケーション起動時に `data/news.json` から配信されたニュースリストをフェッチしてUIにロードする処理を実装する。

## 3. 音声再生プレイヤー (player-interface) の実装
